Project Summary: Reliable UDP Transport Protocol

Project: Plivo Systems Challenge

This report documents the design of a reliable UDP transport protocol developed to handle hostile network conditions, specifically targeting latency constraints and high packet loss environments.

Technical Highlights

  • FEC Strategy: Implemented a fractional XOR parity scheme (FEC-80) to balance bandwidth usage against recovery speed.
  • ARQ Strategy: Developed an event-driven NACK system with Triple-Tap redundancy and congestion-aware cooldowns.
  • Validation: The protocol achieved "VALID" status on Profile A (60ms target) and Profile B (100ms target) while maintaining bandwidth overhead under 2.00x.
